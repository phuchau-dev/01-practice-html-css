Làm quen với html Hyper Text Markup Language

dòng 1 là !DOCTYPE html

dòng 2 là thẻ html lang = "en"

thẻ html sẽ bao tấc cả các thẻ bên ngoài

tiếp theo là thẻ head là phần đầu chứa các link title meta

trong thẻ head sẽ có các thẻ như là title và link meta

tiếp theo là thẻ title là cái tên tiêu đề của trang ở trên chorme có nó nằm ở trên tab

thẻ meta chứa charset='utf-8' đảm bảo hiển thị đúng các ký tự và đa ngôn ngữ

thẻ link sẽ chứa các đường link như là css font chử

tiếp theo là thẻ body thẻ body sẽ chứa toàn bộ nội dung của trang web header content foder sidebar ....

thẻ div là 1 thẻ để nhóm các thẻ khác lại để dể quản lý
